# Analys av Hälsostudie
Detta projekt är en dataanalysuppgift utförd i Python. Syftet är att undersöka ett dataset med hälsodata för att hitta samband mellan livsstilsfaktorer (som rökning och vikt) och hälsoutfall (som blodtryck och sjukdom).

## Branches
Projektet är uppdelat i två delar (branches):
* **del1:** Grundläggande utforskande analys och statistik.
* **del2 (denna branch):** Struktukturerad kod, linjär regression och fördjupad analys.
